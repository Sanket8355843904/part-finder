⚙️ Connect Your Sheet

This app reads directly from your Google Sheet via Apps Script. Follow these steps once:

  1. Open your sheet → Extensions → Apps Script
  2. Paste the doGet() script (provided separately)
  3. Deploy → New Deployment → Web App
  4. Set access to "Anyone", copy the URL
  5. Paste it below 👇

The URL is saved in your browser — you only need to do this once.
